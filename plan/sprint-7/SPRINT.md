## Детально для виконавця

### Принципи mobile-first для фестивалю

- Мінімум тексту на екрані — великі кнопки (touch targets ≥ 48px)
- Без горизонтального скролу
- Критичні дії (підтвердити, взяти, передати) — одним пальцем, одним тапом
- Темна тема за замовчуванням (денне сонце → погано читається світлий фон)

### PWA мета

- Відвідувач встановлює "Festival Menu" на Home Screen → відкривається як аплікація
- Staff встановлює для швидкого доступу до dashboard
- Офлайн-режим: кешоване меню (читання без інтернету)

### Definition of Done

- [ ] Всі сторінки відповідають на mobile без горизонтального скролу
- [ ] Touch targets ≥ 48x48px на критичних кнопках
- [ ] `manifest.json` валідний (перевірити в Chrome DevTools → Application)
- [ ] Service worker кешує меню і статику
- [ ] Офлайн: відвідувач бачить меню з кешу
- [ ] Lighthouse PWA score ≥ 80
- [ ] `uv run pytest -m "tier1 or tier2"` — зелені
